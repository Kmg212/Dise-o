1 Introducci√≥n.

La estad√≠stica inferencial juega un papel crucial en la agronom√≠a al permitirle a los profesionales realizar estimaciones y tomar decisiones basadas en datos limitados. Esta rama de la estad√≠stica aborda:

  1. La estimaci√≥n por intervalos que son herramientas utilizada para construir rangos dentro de los cuales se espera que se encuentre un par√°metro poblacional, como el rendimiento de un cultivo o la eficacia de un tratamiento, proporcionando una medida de precisi√≥n y confianza en los resultados obtenidos a partir de una muestra.

  2. La prueba de hip√≥tesis que es una herramienta fundamental para evaluar afirmaciones o teor√≠as sobre las poblaciones, como determinar si un nuevo fertilizante mejora significativamente el crecimiento de las plantas en comparaci√≥n con un fertilizante est√°ndar.

1.1 Tipos de pruebas estad√≠sticas.

1.1.1 Pruebas param√©tricas y semiparam√©tricas.

Estas pruebas asumen que la distribuci√≥n de los datos siguen un comportamiento en particular.

  • Prueba t para 2 muestras independientes:

    • Prop√≥sito: Compara las medias de dos grupos.

    • Supuestos: Distribuci√≥n normal de los datos de ambas muestras y que las varianzas sean homog√©neas entre ambos grupos (homocedasticidad).

    • Funci√≥n en R: t.test() con argumentos paired=FALSE y var.equal=TRUE .

  • Prueba t welch para 2 muestras independientes:

    • Prop√≥sito: Compara las medias de dos grupos.

    • Supuestos: Distribuci√≥n normal de los datos de ambas muestras y que las varianzas sean diferentes entre ambos grupos (heterocedasticidad).

    • Funci√≥n en R: t.test() con argumentos paired=FALSE y var.equal=FALSE .

  • Prueba t de Student para muestras pareadas:

    • Prop√≥sito: Compara las medias de dos grupos de mediciones relacionadas (e.g., mediciones antes y despu√©s del tratamiento en los mismos sujetos).

    • Supuestos: Normalidad de las diferencias entre las mediciones.

    • Funci√≥n en R: t.test() con argumentos paired=TRUE

1.1.2 Pruebas no param√©tricas

Estas pruebas no asumen una distribuci√≥n de los datos ajustada a la distribuci√≥n normal y son la alternativa a las pruebas param√©tricas y semiparam√©tricas.

  • Prueba de Mann-Whitney Wilcoxon:

    • Prop√≥sito: Compara las medianas de dos grupos.

    • Uso: Alternativa a la prueba t para muestras independientes cuando los datos no son normales.

    • Supuestos: No requiere normalidad, pero los datos deben ser ordinales o continuos.

    • Funci√≥n en R: wilcox.test() con argumento paired=FALSE.

  • Prueba de Wilcoxon:

    • Prop√≥sito: Compara las medianas de dos grupos de mediciones relacionadas.

    • Uso: Alternativa a la prueba t para muestras pareadas cuando los datos no son normales.

    • Supuestos: No requiere normalidad, pero los datos deben ser ordinales o continuos.

    • Funci√≥n en R: wilcox.test() con argumento paired=TRUE.

2.2.2 Paso 2. Verificaci√≥n de supuestos.

El test de normalidad shapiro & wilks contrasta si las observaciones de una muestra aleatoria provino de un poblaci√≥n con distribuci√≥n normal. Este test es potente para probar la normalidad de un conjunto de datos siempre que se tenga m√°ximo 2000 valores muestrales. A diferencia de otros test como el test K-S no permite probar la bondad de ajuste de los datos para distribuciones distintas de la distribuci√≥n normal.

Estad√≠stico de prueba:

\(W=\frac{b^2}{\sum(x_i-\bar{x})^2}\)

Hip√≥tesis estad√≠sticas:

\(H_0:=\) Los datos provienen de una poblaci√≥n que se ajusta a la distribuci√≥n normal.

\(H_1:=\) Los datos provienen de una poblaci√≥n que no se ajusta a la distribuci√≥n normal.

Regla de Interpretaci√≥n.

Valores de p mayores o iguales al nivel de significancia respaldan la \(H_o\). Mientras que valores de p menores al nivel de signficancia respaldan la \(H_1\).

Code 
shapiro.test(FertA)

W = 0.95692, p-value = 0.7392
